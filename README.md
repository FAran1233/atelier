# Atelier — Art Practice Studio

A complete front-end art practice tool. No backend, no accounts — runs entirely in the browser.

## Features

- **Gesture Timer** — countdown with random pose prompts and audio ping
- **Warm-Up Drills** — drawable canvases for lines, circles, ellipses, hatching, boxes
- **Perspective Grid** — 1pt / 2pt / 3pt grids with draggable vanishing points
- **Color Practice** — palette generator and RGB color mixer with scoring
- **Value Scale** — match grayscale targets with instant feedback
- **Practice Log** — sessions, skills, ratings, streak, heatmap, badges
- **Dashboard** — daily challenges, stats, quick access to all tools

## Tech

Plain HTML + CSS + JavaScript. Canvas API for drawing. LocalStorage for persistence.

## Deploy

Deployed via [Vercel](https://vercel.com) — see `vercel.json`.

## Local development

Just open `index.html` in any browser. No build step needed.
